# 🌐 MERN Social Media Web Application

A **full-stack social media web application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This project focuses on real-world social media features such as posts, messaging, user connections, and media sharing, along with modern tools for authentication, background jobs, and image optimization.

🚧 **Status:** Finished

---

## 🚀 Project Overview

This application allows users to connect, communicate, and share content in real time.  
It is designed to strengthen full-stack development skills by implementing scalable architecture, third-party services, and production-level workflows.

---

## ✨ Key Features

### 🔐 Authentication & User Management

- Secure sign-up and sign-in using **Clerk**
- User profile creation and editing
- Protected routes and sessions

### 🧵 Social Features

- Create and share text or image-based posts
- Personalized feed system
- Follow / Unfollow users
- Friend request management
- Story / status uploads
- Search and discover new users

### 💬 Real-Time Communication

- One-to-one real-time chat
- Messaging notifications
- Recent messages overview

### ⚙️ Background Processing

- Background jobs and scheduled tasks using **Inngest**
- Event-driven architecture for async operations

### 🖼 Media Storage & Optimization

- Image upload and delivery using **ImageKit**
- Optimized images for fast performance

---
